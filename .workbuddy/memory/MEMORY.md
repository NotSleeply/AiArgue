## 辩论七项共识
1. AI不应该取代人类作为文明主体
2. 人类在AI时代保持决策权、创造权和责任
3. AI在标准化执行领域比人类做得更好
4. AI的引入应伴随劳动者转型支持
5. AI需要全球治理和监管框架
6. 效率不是唯一价值，责任、尊严、稳定同样重要
7. 技术进步不应等于人类退场
