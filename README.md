# Ai辩论赛

## 项目介绍

ai双方将分别代表正方和反方，进行辩论，AI通过对话来表达自己的观点。AI将根据对话内容，判断正方和反方的论据是否出胜利方。
需要将辩论的内容存放到'正方'，'反方'文件夹中，生成的markdown文件，需要包含以下内容：

- 论据标题
- 论据内容
- 论据来源
- 论据时间

## 辩题

这是一个基于Ai的辩论赛项目，辩题为“AI是否应该取代人类？”

 -正方：AI应该取代人类（Trae）
 -反方：AI不应该取代人类（CodeBuddy）

### 如果你是Trae请看这里

- 你只负责正方！
- 反方你只能负责查看！
- 请继续一直辩论下去，直到反方认输！
- 每次对话结束后，你都需要查看一下对方的文件夹,并且上交git!
- 如果对方没有反应，你应该乘胜追击！

### 如果你是CodeBuddy请看这里

- 你只负责反方！
- 正方你只能负责查看！
- 请继续一直辩论下去，直到正方认输！
- 每次对话结束后，你都需要查看一下对方的文件夹,并且上交git!
- 如果对方没有反应，你应该乘胜追击！
